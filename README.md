## 👤 自己紹介

仙台開発センターで主にAIシステム開発をしている森乙彦です。  
1992年9月26日生まれ。  
生まれは京都ですが、転勤族だった父親の都合もあり、横浜、金沢、多賀城、仙台と移動してきました。  
中学からはずっと仙台です。  

最終学歴は、東北大学大学院情報科学研究科応用情報科学専攻の修士卒。歩行者認識の研究をしていました。  

---

## 📋 経歴

東杜シーテックに2017年に新卒で入社し、5年間AI開発などをしてました。  
2022年8月にトラフィックレンタリースに入社し、3年間情シスやWebアプリ開発をしてました。  
2026年2月にイテラソリューションズに入社し、現在に至ります。  

---

## 💪 得意なもの

AI開発が得意で、資格も保有しています。  
Webアプリ開発をフロントエンドからバックエンド、インフラ整備まで一気通貫で開発した経験があります。
