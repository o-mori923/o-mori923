## 👤 自己紹介

仙台で主にAIシステム開発をしてます。  
1992年生まれ。  
生まれは京都ですが、転勤族だった父親の都合もあり、横浜、金沢、仙台と移動してきました。  
中学からはずっと仙台です。  

最終学歴は、東北大学大学院情報科学研究科の修士卒。

## 📋 経歴

2017年に仙台のシステム開発会社に新卒で入社し、5年間AI開発などをしてました。  
2022年8月にレンタカー会社に入社し、3年間情シスやWebアプリ開発をしてました。  
2026年2月に仙台のシステム開発会社に入社し、現在に至ります。
